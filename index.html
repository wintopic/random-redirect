自建图片 API

欢迎使用自建图片 API！

这个 API 可以从一个包含图片链接的文本文件中随机选择一个图片并返回。你可以使用以下方法来调用 API 并获取图片：

1. 获取 JSON 格式的图片链接

如果你希望获取随机图片的 JSON 响应，可以调用如下接口：

GET https://「your-vercel-project」.vercel.app/api/getImage?type=json

示例响应：

{
  "pic": "https://example.com/image1.jpg"
}

2. 获取图片重定向

如果你希望直接跳转到随机选择的图片链接，可以调用如下接口：

GET https://「your-vercel-project」.vercel.app/api/getImage

浏览器会自动跳转到一个随机的图片链接。

如何使用？

你只需要在你的项目中发起 GET 请求，并根据需求选择返回 JSON 格式的图片链接，或者直接跳转到图片本身。

在请求时，替换「your-vercel-project」为你在 Vercel 部署后生成的项目名称。
