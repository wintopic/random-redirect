图片 API 使用说明

欢迎使用我们的图片 API！通过调用此 API，你可以随机获取一张图片并显示在页面上。

获取 JSON 格式的图片链接 获取随机图片（直接重定向）

API 调用结果将在此显示：

调用方式

你可以使用以下两种方式来调用图片 API：

  • 获取 JSON 格式的图片链接： 访问 /api/getImage?type=json，你将收到一个包含图片链接的 JSON 响应。
  • 直接重定向到随机图片： 访问 /api/getImage，你将被重定向到一张随机图片。

示例请求

https://.vercel.app/api/getImage?type=json https://.vercel.app/api/getImage
